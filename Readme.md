# AdaKerja React Native Task

AdaKerja is a hybrid app for dealing with github repos and searching for commits.

## Setps to run the project
```bash
git clone https://car-wash-web@bitbucket.org/car-wash-web/reactdemo.git
cd reactdemo
npm i expo-cli
npm install
expo start
```
